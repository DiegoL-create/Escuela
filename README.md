# Sistema Escolar

Proyecto desarrollado en PHP y MySQL.

## Requisitos

- PHP 8
- XAMPP
- MySQL

## Instalación

1. Clonar el repositorio.
2. Importar Escuela.sql.
3. Copiar la carpeta en htdocs.
4. Abrir:

http://localhost/SistemaEscuela/LoginE.php
